linux笔记

linux命令

  1. pwd：显示当前完整路径。

  2. cd：更改当前工作目录。

    ..返回上一级。 ~表示home目录

  3. ls：显示当前路径下所有目录和文件。

    “-l”：列出文件详细信息。

    “-a”：显示所有文件(包括隐藏文件)

  4. clear：清屏

  5. rm：删除文件。

    “-r”：删除目录，“-rf”：强制删除文件和目录

  6. touch：创建文件

  7. mkdir：创建目录。

    “-p”：可以一次创建多级目录。

  8. df：查看磁盘当前可用空间(查看当前光盘挂载情况)。

  9. free：查看当前可用内存容量。

  10. file：输出文件内容的简要描述。

  11. cp：复制文件或目录。

  12. mv：移动或重命名文件或目录。

  13. 输出重定向。>：覆盖输出，>>：追加输出。语法：指令>文件路径

  14. cat：直接打开文件，查看文件内容。

  15. head：查看文件前n行。

    语法：head -n 文件路径(n为行数，默认为前十行)

  16. tail：查看文件后n行。

    语法：tail -n 文件路径（n为行数，默认后十行）

    tail -f 文件：查看文件的动态变化内容（不能是用户手动增加的）

  17. less：查看文件内容以较少内容输出(可以翻页)，more类似(可以自动退出)

  18. wc：统计文件内容信息。

    参数 说明
    -l 表示lines行数
    -w 表示words单词数
    -c 表示bytes字节数

  19. grep：查找指定内容

  20. |：管道符

    语法：命令1 | 命令2 (命令1的结果作为命令2的输入)。

    案例：ls / |grep y (查询根目录下包含y字母的文档名称)

  21. ln：创建链接(快捷方式)

    语法：ln 参数 源文件 新文件

    参数：-s：软连接(不加为硬链接)

  22. alias：别名

    语法：alias 别名=“可执行的命令”

  23. su：切换用户

  24. useradd：创建用户(在root下)

    语法：useradd 参数 用户名

    参数 说明 语法
    -u 指定用户UID useradd -u 用户UID 用户名
    -g 指定用户基本组 useradd -g 组名 用户名
    -G 指定用户附加组 useradd -G 组名 用户名
    -e 指定用户失效时间 useradd -e yyyy-mm-dd(失效时间) 用户名
    -d 指定用户家目录 useradd -d 家目录 用户名

  25. groupadd：创建组

    语法：groupadd 组名

  26. userdel：删除用户

    语法：userdel 参数 用户名(参数：-r：删除用户及家目录)

  27. groupdel：删除组。

  28. usermod：修改用户信息

    语法：usermod 参数 用户名

    参数 说明 语法
    -l 更改用户名 usermod -l 新用户名 旧用户名
    -u 更改用户UID usermod -u 新UID 用户名
    -g 更改用户基本组 usermod -g 组名 用户名
    -G 更改用户附加组 usermod -G 组名 用户名
    -a 追加用户附加组 usermod -a -G 新附加组 用户名

  29. passwd：修改密码

    参数 说明
    -l 锁定用户
    -u 解锁用户

  30. gpasswd：移除组内用户

    语法：gpasswd 参数 用户名

    参数 说明
    -d 删除
    -a 添加（为附加组）

  31. chmod：修改权限

    语法：chmod 参数 权限模式 文件（目录）

    权限模式字母形式 数字形式
    u：用户 +：增加权限 r：可读 4：读
    g：所属组 -：减少权限 w：可写 2：写
    o：其他人 =：确定权限 x：可执行 1：执行
    a：全部 0：无权限

  32. chown：更改文档所属用户。

    语法：chown 参数 新用户:组名 对象（不修改可以不填）

    参数：-R：递归(目录及目录下所有文件)

  33. setfacl：给予指定用户某个对象特殊权限。

    语法：setfacl 参数 设定值 对象

    参数 说明 语法
    -m 设定规则 setfacl -m 身份:用户:权限 对象（身份：用户：u，组：g）
    -x 取消特殊权限 setfacl -x 身份:用户 对象
    -b 取消所有特殊权限 setfacl -b 对象

    例子：给zs已a.txt文件读和写的权限

  34. sudo：让管理员实现定义某些命令谁可以执行。

    先用管理员root输入visudo进入配置文件写上

  35. which：查看指定命令的完整路径。

  36. ps：显示系统执行的进程。

    参数 说明
    -a 显示当前终端所有进程信息
    -u 以用户的格式显示进程信息
    -x 显示后台进程进行的参数

  37. pstree：查看进程树。

    参数 说明
    -p 显示进程的PID
    -u 显示进程所属用户

  38. kill：终止进程。

    语法：kill 参数 进程号(PID)

    killall 进程名称

    参数 说明
    -9 强制停止
    -15 正常结束进程(并回收资源)

  39. find：查找文件或目录

    语法：find 指定路径 参数(参数值) (后续处理操作)

    1. 简单查找：find 指定路径
    2. 根据类型查找。

      语法：find 路径 -type 参数

    参数 说明
    -type d 显示路径下所有目录
    -type f 显示路径下所有文件
    -type l 显示路径下所有链接文件
    1. 根据深度查找。
      语法：find 路径 参数
    参数 说明
    -maxdepth n 最大查找深度
    -mindepth n 最小查找深度
    -maxdepth n1 -mindepth n2 两个一起用
    1. 根据身份查找。
    参数 说明
    -user 用户名(UID) 根据用户查找
    -group 组名(GID) 根据组查找
    -nouser 查找无主用户
    1. 根据文件名查找：-name 字段
    2. find的逻辑组合
    参数 说明
    -a 与逻辑
    -o 或逻辑
    -not或！ 非逻辑
    1. 后续处理动作
    参数 说明
    -print 把命令输出到终端上
    -ls 在指令最后面加上可以显示具体信息
    -delete 把找到的东西删除
    -ok 语法：-ok 可执行的命令 {find的命令找到后的结果} ;

  40. gzip：对单一我呢见进行压缩(生成*.gz压缩文件)压缩后源文件不保留

    语法：gzip 参数 被压缩文件

    参数 说明 语法
    -d 解压缩(和gunzip命令一样) gzip -d 压缩文件.gzip
    -c 对文件压缩并保留源文件，可以进行重命名 gzip -c 被压缩文件 > 压缩文件名.gz
    -k 和-c用法一样(centos8支持) gzip -k 被压缩文件
    -# #为1-9数字，数字越大压缩比越大，默认为6

  41. zcat：在不解压缩文件的前提下查看文件内容(xzcat和bzcat)

  42. bzip2：和gzip效果一样但后缀为.bz2

    参数 说明
    -k 压缩后保留源文件(gzip没有)，其他参数和gzip一样。

  43. xz：和bzip2、gzip效果一样。

    后缀为.xz。解压缩命令为unxz

  44. zip：压缩文件或目录(压缩后默认保留源文件)

    1. 直接压缩文件

      语法：zip 压缩文件名(*.zip) 被压缩文件

      压缩后保留源文件，可修改文件名，可指定压缩到具体路径

    2. 解压缩文件

      语法：unzip *.zip

      1. 注意是否会生成重名文件，根据实际情况，替换或重命名

      2. 参数：-d 指定到某路径 unzip *.zip -d 指定的路径

      3. 压缩目录：zip 参数 压缩目录名(*.zip) 被压缩目录
        不加参数只对单一的目录本身压缩，目录下内容没有压缩
        参数：-r 对目录及目录下内容整体压缩

      4. 解压缩目录：unzip *.zip

        1. 注意是否有重名目录

        2. 指定解压缩路径：unzip *.zip -d 指定路径

        3. 在指定路径的同时进行重命名：unzip *.zip -d 指定路径/.zip

        4. 如果只想压缩目录下内容(不包含目录本身)，先进入目录，在目录下操作。

  45. rpm(目前只用查询软件安装版本)

    语法：rpm -q 具体命令

  46. yum：包的管理

    1. 找仓库路径

      注意：要配置本地仓库的话，一定要挂载到一个目录下

    2. 网络仓库：

      1. 阿里云仓库：https://mirrors.aliyun.com/

      2. 华为云仓库：https://mirrors.huaweicloud.com/

      3. 搜狐云仓库：https://mirrors.sohu.com

      4. 网易云仓库：https://mirrors.163.com

      5. epel仓库的网络地址[https://mirrors.aliyun.com/epel7 x86 64/](https://mirrors.aliyun.com/epel7 x86 64/)

      6. epel的安全密钥地址：https://mirrors.aliyun.com/epel/RPM-GPG-KEY-EPEL-7

    3. 配置网络仓库文件

      仓库文件名一定是*.repo

    4. 配置本地仓库文件

      1. df：查看当前光盘挂载情况

      2. mount：挂载

        语法：mount /dev/cdrom /media

        #将光盘挂载到/media目录下

      3. 仓库文件*.repo

    5. yum的使用

      语法：yum 选项 参数

      CentOS 7 - yum常见命令_yum provider-CSDN博客

      选项 说明 参数 说明
      -h 显示帮助信息 install 安装rpm软件包
      -y 对所有的提问都回答“yes” update 更新rpm软件包
      -c 指定配置文件 check-update 检查是否有可用的更新rpm软件包
      -q 安静模式 remove 删除指定的rpm软件包
      -v 详细模式 list 显示软件包的信息
      -d 设置调试等级(0-10) search 检查软件包的信息
      -e 设置错误等级(0-10) info 显示指定的rpm软件包的描述信息和概要信息

    | -R | 设置yum处理一个命令的最大等待时间 | clean | 清理yum过期的缓存 |
    | -C | 完全从缓存中运行，而不去下载或者更新任何头文件 | shell | 进入yum的shell提示符 |
    | | | resolvedep | 显示rpm软件包的依赖关系 |
    | | | localinstall | 安装本地的rpm软件包 |
    | | | localupdate | 显示本地rpm软件包进行更新 |
    | | | deplist | 显示rpm软件包的所有依赖关系 |

    1. yum list 包名

      #查看仓库有无指定安装包

    2. yum remove 包名

      #卸载指定包（一般不用）（centos7只卸载包，centos8会把依赖包一并卸载）

    3. yum history

      #查看安装记录

    4. yum history info n

      #查看第n条安装历史

    5. yum history undo n

      #撤销第n步操作

    6. yum history redo n

      #重做第n步操作

  47. tar：文件目录的打包与压缩

    语法：tar 参数 路径/*.tar *(被打包对象)

    1. 打包

      语法：tar -cf 路径/*.tar *(被打包对象)

      参数 说明
      -c 打包
      -f 通用参数
      -v 显示信息
    2. 解包(解压缩)

      语法：tar -xf 路径/*.tar

      参数 说明
      -x 解包
      -C 解包到指定路径：tar -xf 路径/*.tar -C 指定路径
    3. 打包并压缩

      语法：tar -zcf 路径/*.tar.gz 被打包文件

      参数 说明
      -z 调用gzip命令
      -j 调用bzip2命令
      -J 调用xz命令
      -a 自动根据压缩文件后缀选择压缩方式

快捷键

  1. ctrl + c：强制停止
  2. Ctrl+d：退出登出
  3. Ctrl+r：搜索历史命令
  4. Ctrl+a/e：光标移动到命令开始或结束
  5. Clear或ctrl+l：清屏
  6. History：查看历史命令

Vim文本编辑器

  1. vim 文件：打开文件

  2. vim 数字 文件：打开文件并将光标移到n行

  3. vim /关键词 文件：打开文件并高亮关键词

  4. shift + 6或者^：光标移到行首

  5. shift +4或者$：光标移到行尾

  6. gg：光标移到首行。Pgup：向上翻屏

  7. G：光标移到末行。Pgdn：向下翻屏

  8. 复制光标所在行：按键：yy 粘贴：p

  9. 光标所在行及向下复制行数：按键：数字yy

  10. 剪切或删除光标所在行：按键：dd（删除后下一行上移）

  11. 剪切或删除光标所在行及向下指定行数：按键：数字dd（删除后下一行上移）

  12. 删除后当前行变成空白行：按键：D

  13. 撤销或恢复：按键：D 恢复：ctrl+r

  14. 将光标移到指定行：按键：数字+G

  15. 以当前光标为准，向上或向下移动n行。按键：数字+pgup或pgdn

  16. 末行模式下快速移动：数字

  17. 末行模式下：

    保存 :w 。另存为 :w 路径 。退出 :q 。 保存并退出 :wq或x 。 强制 ！ 。查找 :/关键词 （上一个/下一个 N/n）

  18. 显示行号（临时）
