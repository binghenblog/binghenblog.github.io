Microsoft 激活脚本 （MAS）

开源 Windows 和 Office 激活器，具有 HWID、Ohook、TSforge、KMS38 和在线 KMS 激活方法，以及高级故障排除。

如何激活 Windows / Office？

方法 1 - PowerShell（Windows 8 及更高版本） ❤️

信息

  1. 打开 PowerShell
    为此，请按 Windows 键 + X，然后选择 PowerShell 或终端。

或者，您可以使用以下 （这将在将来弃用）：

  1. 您将看到激活选项。
    按 [1] HWID 进行 Windows 激活。
    按 [2] Ohook for Office 激活。

方法 2 - 传统（Windows 7 及更高版本）

  1. 使用以下链接之一下载文件：

  2. 右键单击下载的 zip 文件并解压。
  3. 在解压缩的文件夹中，找到名为 的文件夹。All-In-One-Version
  4. 运行名为 的文件。MAS_AIO.cmd
  5. 您将看到激活选项。按照屏幕上的说明进行作。

不工作 ❓

  • 如果您无法使用 PowerShell 方法启动 MAS，请参考上面的方法 2。
  • 如果 MAS 启动但显示错误，请检查以蓝色突出显示的故障排除步骤并按照这些步骤作。
  • 如果问题仍然存在，请随时在此处与我们联系。

注意

  • PowerShell 中的 IRM 命令从指定的 URL 下载脚本，然后 IEX 命令执行该脚本。
  • 在执行命令之前，请始终仔细检查 URL，并在手动下载文件时验证来源是否可信。
  • 请小心，因为有些人会通过更改 IRM 命令中的 URL 来传播伪装成 MAS 的恶意软件。

MAS 最新版本

最新版本 - v3.0（2025 年 2 月 14 日）
GitHub / Azure DevOps/自托管 Git

特征

  • HWID （数字许可证）永久激活 Windows 的方法
  • 奥胡克永久激活 Office 的方法
  • TSforge 公司永久激活 Windows/ESU/Office 的方法
  • KMS38 系列激活 Windows 直到 2038 年的方法
  • 在线 KMS激活 Windows/Office 180 天的方法（续订任务终身）
  • 高级激活疑难解答
  • $OEM$ 个用于预激活的文件夹
  • 更改 Windows 版本
  • 更改 Office 版本
  • 检查 Windows/Office 激活状态
  • 提供 All In One 和 Separate Files 版本
  • 完全开源且基于批处理脚本
  • 更少的防病毒检测
