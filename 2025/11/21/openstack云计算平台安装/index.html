openstack云计算平台搭建与管理

冰痕2025-11-212025-11-22

前提准备：

  1. 系统镜像：openEuler-22.03-LTS-SP3-x86_64-dvd.iso

  2. 本地仓库：openStack-train.iso

  3. 测试镜像：cirros-0.5.1-x86_64-disk.img

  4. 远程控制软件：MobaXterm

  5. 虚拟机：VMware17.6.0

1.安装配置VMware虚拟机

创建新的虚拟机

选择自定义(高级)

这边选择稍后安装操作系统

选择处理器内核数量可以多给点

虚拟机内存至少给4GB，如果一开始给的内存少的话，后面可以再添加

选择仅主机模式，后面还要再添加一张网卡(NAT模式)

选择将虚拟磁盘存储为单个文件，最大磁盘大小我这是给的100GB，实际可以根据需求扩大

这边选择自定义硬件，来添加新网卡和打开CPU的虚拟化

创建完成以后，在VMware虚拟机的左上角选择编辑，虚拟网络编辑器

将仅主机的子网地址改为192.168.10.0/24

将NAT的子网地址改为192.168.20.0/24

将这俩的DHCP打开

开始挂载镜像文件

挂载上之后打开虚拟机，用键盘的上键选择第一个Install openEuler 22.03-LTS-SP3

刚进来选择中文-简体中文，选择右下角继续

选择网络和主机名，把这两个网卡ens33和ens34打开，然后下面有主机名输入controller点击应用

然后配置root账户密码：openstack0#，也可以设置成其他的

都配置完成后，点击开始安装

等待安装完成选择重启系统

当前界面是等待登录界面

输入用户名root和之前设置的密码openstack0#

登录成功就显示这个页面

5.虚拟网络，云主机搭建

5.1 配置虚拟网络

先把VMware的DHCP关闭

安装网桥管理工具包(俩节点都安装)

1. 用Dashboard创建和管理虚拟网络和子网

打开浏览器输入192.168.10.101(计算节点的IP地址)

选择管理员 -> 网络 -> 网络 ->创建网络

网络名称vir-net(随便起)项目选project，供应商Flat，物理网络provider(和ml2_conf.ini中的flat_networks = provider保持一致)，选择共享的和外部网络复选框

子网名称subnet(随便起)，网络地址192.168.20.0/24，网关192.168.20.2

配置IP地址池和DNS服务器

配置好后点创建就可以看到虚拟网络了

2. 用命令模式创建和管理虚拟网络和子网

由于只能创建一个Flat网络，需要将先前的虚拟网络删除(没创建的就可以跳过此步骤)

5.4 用云镜像部署云主机

1. 检查系统环境

第一步，下载系统云镜像

第二步，检查时间同步服务

差距过大的话需要对时间机型同步，可以用Chrony时间同步服务进行自动同步外，还可用date命令进行时间的手动同步

第三步，检查现有云主机

如果有可以关闭

2. 部署CentOS云主机

打开浏览器输入192.168.10.101(计算节点的IP地址)

选择项目 -> 计算 -> 镜像 -> 创建镜像

镜像名称随便写，镜像源选择CentOS-7-x86_64-GenericCloud-2009.qcow2，镜像格式选择QCOW2-QEMU模拟器

选择项目 -> 网络 -> 安全组 -> 管理规则

选择添加规则，主要是添加两条规则：SSH和ping

创建密钥对

选择项目 -> 计算 -> 密钥对 -> 创建密钥对

创建密钥对后，下载密钥对文件

创建实例

实例名称随便

选择不创建新卷，镜像选择centOS7

这是定制化脚本

ssh_pwauth：是否允许SSH协议用密码登陆

expire：配置密码是否要过期，False表示永不过期

CentOS7的默认用户是centos

默认用户的密码设置为000000，用户root密码000000
